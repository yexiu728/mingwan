css书写与选择器 书写方式 1、内嵌式：代码厚重，只能用于当前页面 初学者使用 2、行内式-写在标签内：代码厚重，只能用于当前标签 特殊情况下使用 3、外联式：可以复用 创建css文件，使用link标签引入 选择器 1、通配符选择器 0，0，0，0 范围广，匹配所有元素 * { 属性名：属性值 } 2、标签选择器 0，0，0，1 可以给指定的标签设置样式，不可以分别指定 标签名 { 属性名：属性值 } 3、类选择器 0，0，1，0 可以给指定的标签设置样式，一个标签可以有多个类样式，类名之间用空格隔开 .类名 { 属性名：属性值 } 4、id选择器 0，1，0，0 只能出现一次，不能重复 #id名 { 属性名：属性值 } // 行内样式 1，0，0，0 // !important 无穷大 命名规范： 少用大写，使用驼峰命名法 不能以数字开头，不要使用中文 字体设置 内外边距 内边距：padding 外边距：margin 简写时设置方向：上右下左 外边距塌陷（父子关系，兄弟关系） 相邻块元素垂直外边距合并：无法解决，只能避免 嵌套块元素垂直外边距合并：给父元素定义1px边框，或者父元素设置 overflow: hidden 复合选择器 后代：空格 div .clazz {} 子代：大于号 .clazz>div {} 交集：连接 div.clazz {} 并集：逗号 .clazz1, .clazz2 {} 伪类：冒号 .clazz:hover {} 属性：中括号 .clazz[key=value] {} 显示模式 display inline 行内元素，一行可以显示多个，不能设置宽高 block 块级元素，一行只能显示一个，可以设置宽高 inline-block 行内块，一行可以显示多个，可以设置宽高 背景 background-color 背景颜色 background-image 背景图片地址 background-repeat 是否平铺 background-position 背景位置 background-attachment 背景固定还是滚动 background-size 背景图片大小 背景的合写（复合属性） background:背景颜色 背景图片地址 背景平铺 背景滚动 背景位置 / 背景图片大小 伪类 单冒号 : :link /* 未访问的链接 */ 只对a标签有效 :visited /* 已访问的链接 */ 只对a标签有效 :hover /* 鼠标移动到链接上 */ 几乎对所有标签有效 :active /* 选定的链接 */ 几乎对所有标签有效 伪元素 双冒号 :: 1. E::before 开始 和E::after 结束 结构(位置)伪类选择器（CSS3) :first-child :选取属于其父元素的首个子元素的指定选择器 :last-child :选取属于其父元素的最后一个子元素的指定选择器 :nth-child(n) ：匹配属于其父元素的第 N 个子元素，不论元素的类型 :nth-last-child(n) ：选择器匹配属于其元素的第 N 个子元素的每个元素 不论元素的类型，从最后一个子元 素开始计数。 n 可以是数字、关键词或公式 盒子阴影 box-shadow: x轴 y轴 模糊距离 [阴影尺寸] 阴影颜色 浮动 float：left 或 right，脱标 定位 静态定位-默认 相对定位： 根据自己定位，不脱标 绝对定位： 脱标 父元素有定位，根据父元素为定位 父元素没有定位，会找子元素最近的定位来定位 父元素都没有定位，以body来定位 子绝父相：父元素为相对定位时，子元素根据父元素定位 子绝父绝：父元素为绝对定位时，子元素根据父元素定位 固定定位 脱标，固定位置显示在屏幕 使块级元素有行内块元素的特点，一行可以显示多个 使行内元素有块级元素的特点，可以设置宽高 z-index 只能用在相对定位，绝对定位和固定定位，浮动和静态定位不行 值默认是0，值越大越在上方
