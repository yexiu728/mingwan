Math对象的方法的作用

  方法名 	            作用    	备注
  Math.random()    	随机数 * 	不需要传参
  Math.round(x)        	四舍五入取整
  Math.floor(x)    	    向下取整
  Math.ceil()           	向上取整
  Math.max(x,y...)    	最大值
  Math.min(x,y)         	最小值
  Math.abs(x)       	    绝对值
  Math.pow(x,y)       	次幂    	     x的y次幂

获取 0~n 随机整数

    Math.floor(Math.random() * n)

函数基本结构

    1、函数名
    2、函数参数
    3、函数返回值

函数定义：

    function 函数名(参数) {
        函数体;
        return 返回值;
    }

    function sum(num1 = 10, num2 = 20) {
        return num1 + num2;
    }

函数调用：

    函数名();
    sum();

函数三要素

    1. 函数名
    2. 参数
    3. 返回值

函数名命名规则

    必须遵守：
    	1. 不能以数字开头
    	2. 特殊字符只能使用美元符$ 和下划线 _
    	3. 不能使用关键字和保留字
    建议遵守：
    	4. 使用驼峰命名法
    	5. 名字要有语义化

形参和实参

    形参：
        声明函数时括号内的参数就是形参
        1、参数的本质是变量，但是函数参数不要写var
        2、多个参数之间用英文状态下的逗号隔开，不能用其他符号
        3、参数可以赋一个默认值，但是调用的时候传递了参数，默认值就会被覆盖
        4、一般情况下，参数调用的时候和定义的时候的个数是一样的
    实参：
        调用函数时传入的参数值
